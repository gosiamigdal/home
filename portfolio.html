Trip planner

SimpleSanFrancisco lets you create personalized guide which you can easily use during your San Francisco trip. It has weather and Facebook APIs integrated. It was build using: Flask, Python, HTML5, CSS3, Twitter Bootstrap, JavaScript, SQLAlchemy and Sqlite.

Events finder

Findevents allows you to easily find events for your free time. I uses your primary Google Calendar. It is Beta version, so it is still lots to work on. It has Eventbrite API integrated. It was build using: HTML5, CSS3, JavaScript and Twitter Bootstrap.
